# Install

- Run `npm install`
- Create `src/config/firebase.ts` file and store

```
export const firebaseConfig = {
  apiKey: '',
  authDomain: '',
  projectId: '',
  storageBucket: '',
  messagingSenderId: '',
  appId: '',
}

export const firebaseIosClientId = ''
```

Find more information about this [here](https://docs.expo.io/versions/latest/sdk/google-sign-in/)

# Run

Run `npm start`
